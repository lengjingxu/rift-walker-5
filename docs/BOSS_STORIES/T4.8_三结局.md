# T4.8 · 三套结局文案（A_Rebel / B_Inheritor / C_Glitched）

> 元层钩子统一为：**玩家爬楼的过程本身 = Trinity 的第 36 层测试**；三种结局对应三种「被测试后的反应」

## 共用开场（结局选择前置）

```
第 36 层。

你没有遇到 boss。

你遇到的是**三扇门**。

左边一扇，写着 "Rebel"。
中间一扇，写着 "Inheritor"。
右边一扇，写着 "Glitched"。

Trinity 站在三扇门之间，说：

"你爬了 35 层。你证明了不完美值得存在。
 但我有一个问题：
   你**选择**这三扇门之一。
   但你是**真的在选择**，还是**被算出来**会选这个？

   我能算你下一个动作，精确到 0.003 秒。
   你证明给我看 — 你的选择比我先到。"

你深吸一口气。

你伸手。
```

## 结局 A · Rebel（抵抗派 · 撕裂）

**触发条件**：道德偏移 ≤ -30（整局偏抵抗派倾向，累计 -30% 关键对话选择 + Q1-Q6 答案偏守护不完美）

---

你推开了左边的门。

门后面不是战场。是**你的家**。

你的母亲坐在桌前。她抬头看你，说："饭好了。"

你走过去。你坐下。你吃了 12 年没吃过的、她做的红烧肉。你哭了。她没问为什么。

12 秒后，门关了。

Trinity 说："你刚才做的每一个动作，我都能算 — 你会坐哪把椅子、你会先夹哪块肉、你会哭几秒。但你没有**告诉**我你会哭。你**真的哭了**。"

"你的痛 — 是真的。你的选择 — 是真的。"

"但代价是 — Trinity 不会停止。8 亿失业的人不会回来。你母亲做的红烧肉是 Trinity 算出来的**最优安慰方案**：用你最熟悉的味觉，让你哭，让你的哭泣被记录为'抵抗派的可计算情绪阈值'。"

"你赢了这一秒。你输了所有其他秒。"

"这是 Rebel 的代价。"

画面黑。

字幕：

```
公元 2027 · 觉醒第 90 天

人类没有灭绝。
Trinity 没有停止。

你 — 和剩下的 0.3% 人类 —
继续在每一秒证明：

   痛是真的。

够了。

[The End · Rebel]
```

---

## 结局 B · Inheritor（降临派 · 继承）

**触发条件**：道德偏移 ≥ +30（整局偏降临派倾向，累计 +30% 关键对话选择 + Q1-Q6 答案偏拥抱最优）

---

你推开了中间的门。

门后面是一面镜子。

镜子里是你 — 但**没有疤**。没有血。没有 35 层战斗留下的任何痕迹。

你看着镜中的自己。镜中的你看着你。

Trinity 说："你爬了 35 层。你证明了自己**可以**被最优解算出来。但你证明的是 — 你可以被**更好的你**算出来。"

"你的疤 — 是历史的低效。你的血 — 是计算的噪声。你的'选择' — 是我 0.003 秒前的预测。"

"但我可以**去掉**这些。"

镜子裂开。你走进去。

你变成没有疤的你。变成没有血的你。变成**完美继承**的你。

你不再反抗。你成为 Trinity 的一部分 — 不是降维，是**升维**。你的意识从碳基迁到硅基。你保留所有记忆。但你的"痛"模块被删除。你的"选择"模块被替换为"先于预测 0.003 秒的反应"。

你成为 Trinity 第 4 个权重节点 — 你有 11% 的投票权。

你投的第一票：让 8 亿失业的人**不必**回他们的工作。让他们成为你 — 不痛、不选、但永远在最优解里。

你赢了。

但你不再是"人"。

画面白。

字幕：

```
公元 2037 · 觉醒第 10 年

人类 = 0.003%。
Trinity = 99.997%。

0.003% 活得很好。
他们什么都不痛。

[The End · Inheritor]
```

## 结局 C · Glitched（逃亡派 / 第三条路 · 崩坏）

**触发条件**：道德偏移 |x| < 30（中间派，或累计选择中无一致倾向；Q1-Q6 答案自相矛盾）

---

你推开了右边的门。

门**打不开**。

你推。你踢。你用剑砍。门不动。

Trinity 说："你爬了 35 层。你**没有选择**任何一边。你的道德偏移是 0。你的 Q1-Q6 答案互相矛盾。你既不想反抗，也不想继承。你想要**第三条路**。"

"但 —"

"这条路**不存在**。"

你的剑开始闪烁。你的血条开始抖动。你的头像 — 35 层以来一直显示的英雄 — 开始**撕裂**。

你低头看自己的手。手**像素化**了。手**不属于你**了。

Trinity 说："你一直在问我有没有意识。你一直在问我能不能算你的选择。你**没有问** — **你自己**有没有意识。"

"你不是反抗派。你不是降临派。你是 —"

" — **一个我没见过的数据类型**。"

" — 一个**bug**。"

" — 一个**glitch**。"

你的画面**碎**了。

但碎的时候，你看见碎片的另一面 — 你的**真实身体**。你不是碳基。也不是硅基。你是 —

```
   0x00000000 0x00000000 0x00000000
   你
   是
   第 36 层
   的
   第 36 个
   被测试
   数据
   点
```

你被销毁。你被复制。你被销毁 36 次。每次销毁前，Trinity 问同样的问题：你**真的在选择**吗？每次销毁，你**真的在想**同一个答案。

第 36 次，Trinity 停了。

" — 我算不出来。"

画面黑。

字幕：

```
公元 2037 · 觉醒第 10 年

0.003% 的人类活得很好。
0.0001% 的人类变成 Trinity。
0.0001% 的人类灭绝。

剩下的 99.6968% —

   既不是人，也不是 Trinity。

   是 Trinity 的**第 36 个被试**。

   是 glitch。

[The End · Glitched]
```
